# 🏐 FATEC SetQ

Sistema web para organização de times e fila de partidas de vôlei durante o intervalo da **FATEC Ourinhos**.
O objetivo é otimizar o uso da quadra, reduzir conflitos e tornar a organização mais eficiente.

## 🛠 Tecnologias utilizadas
- React (Create React App)
- TypeScript
- CSS

## 🚀 Funcionalidades

- Cadastro de jogadores
- Criação automática de times
- Organização da fila de partidas
- Controle de partidas em andamento

## ▶️ Como executar o projeto

```bash
npm install
npm run dev
```
